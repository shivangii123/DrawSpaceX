# DrawSpace

DrawSpace is a browser-based drawing application inspired by tools like Excalidraw.
It allows users to create and manipulate shapes on an HTML Canvas using JavaScript.

## Features

- Draw rectangles
- Draw lines
- Draw triangles
- Draw circles
- Shape-based rendering system
- Shape selection
- Dynamic rendering using the Canvas API

## Tech Stack

- HTML
- CSS
- JavaScript
- Canvas API

## Project Goal

The goal of DrawSpace is to understand how modern drawing tools work internally by
building a mini whiteboard system from scratch.

## Future Improvements

- Drag and drop shapes
- Resize shapes
- Delete shapes
- Layer ordering (bring to front / send back)
- Undo / Redo system
- Export drawing as image (PNG / SVG)
- Grid snapping
- Keyboard shortcuts
- Multi-shape selection
